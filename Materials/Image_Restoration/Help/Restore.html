Restore

This is one of the main applications of this program. Just give some inputs and we will try to restore it.


Required inputs

You will need an image which you wish to restore. You need to provide the following parameters:
  • Type of filter
  • Blur length
  • Blur angle
  • Additional parameters depending on the type of filter
Click Here to know what these parameters mean.

Note:
  • Use the 'Estimate' button to let us estimate what values are right for your image to restore.
  • The angle estimate button returns 10 highly probable blur angles. Use the 'Next' button to go through the possible values.


Types of Filters

  • Inverse Filter:
    This is a very basic restoration filter. It does not require any other parameter. This filter generally gives poor results thus we do not recommend it.
  • Wiener Filter:
    This filter gives descent results. Its quite fast too. It requires Signal to Noise Ratio (SNR). Lower the SNR, better the results.

    Note: Use the 'Estimate SNR' button to let us recommend you some value.
  • Lucy - Richardson:
    This filter is one of the most robust filters. Its immune to almost everything. Since it is an iterative filter, it requires the no. of iterations.

    Note: Increasing the iterations significantly increases the amount of time taken to restore an image. Keep no. of iterations to approximately 100.
  • Compare Filters:
    You can compare the results of the different filters. We provide an indepth report to help you assist in selection of the right filter. Comparison is done using some predefined methods.

How to Restore?

Just follow these steps to restore an image:
  1. Open a degraded image.
  2. Input the blur parameters 'Length' & 'Angle'. Use the slider or the text field to enter these parameters.
  3. Select the additional parameters if needed.
  4. Click on 'Restore Image' button.

Tips:
  • Use the 'Estimate' button to let us estimate what values are right for your image to restore.
  • You can also save the degraded image from the File->Save.
Note:
  • Our estimation may go wrong. They are approximate values. So for efficient restoration try other values too.
  • This program can work only on gray scale image. Incase you input a color image, it will be converted to gray scale image.
  • There is the 'Expert Mode' which you can use once you get an idea of how the estimation algorithms work.
    • 'Expert Mode' in angle estimation displays the edges in the image. From there you can say whats the blur angle.
    • 'Expert Mode' in length estimation shows a plot. The first negative value is the blur length.
