Getting Started

What we got for you?

This program can degrade as well as restore an already degraded image. It hides all the complexity from the user to give a good user environment.

You dont need to be an expert in order to use this program. This program is meant for people like you and me. No past knowledge in image degradation or image processing is required. Whatever is required is explained here.

Read this manual to know the basics of it.


Where do I start?

Use the 'Filter' menu to select your choice. You can start by either degrading an image or restoring an already degraded image.

We have provided help on all the pages. Incase you have any trouble just use 'Help' button.


Menu

File: You can do operations like open and save images from here.

  • Load Degraded: Load a recently degraded image.
  • Load Restored: Load a recently restored image.
  • Open: Open a file stored in memory.
  • Save: Save a degraded/restored image.
Edit: Operations related to the input image are provided here.

  • Undo: Undo operations performed on the input image. e.g.: Crop
  • Redo: Redo operations performed on the input image. e.g.: Crop
  • Crop: Select a part of the input image for degradation/restoration. Once selected, press the mouse button on the starting and the ending point of the image. Do not drag the mouse.
  • Zoom: Zoom a part of the restored image. It enlarges a part of the image and displays on a separate area on the GUI. To zoom a part of the image, just click on the desired location after selecting this option from 'Edit' menu.
Note: To come out of 'Crop' and 'Zoom' press 'Esc'.

Filter: Select the desired filter from here. You can either degrade an image or restore an image.

  • Degrade: Select this option to degrade an image. Click here to learn more about degradation.


  • Restore: Select this option to restore an image. You can select any of the 3 filters. A comparison chart is prepared to help you select the right filter. Click here to learn more about restoration.
