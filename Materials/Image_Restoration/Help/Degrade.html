Degrade

You can degrade an image using this software. Just give some inputs and you could degrade the image the way you want.


Required inputs

You will need an image which you wish to degrade. You need to provide the following parameters:
  • Blur length
  • Blur angle
  • Type of noise (if needed)
    • Salt & Pepper
    • Gaussian
    • Poisson
    • Speckle
Click Here to know what these parameters mean.


How to Degrade?

Just follow these steps to degrade an image:
  1. Open an image.
  2. Input the blur parameters 'Length' & 'Angle'. Use the slider or the text field to enter these parameters.
  3. Select noise type if you want.
  4. Click on 'Degrade Image' button.

Tips: You can also save the degraded image from the File->Save.

Note: This program can work only on gray scale image. Incase you input a color image, it will be converted to gray scale image.
